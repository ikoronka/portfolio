# Portfolio

A modern, bilingual personal portfolio website built with React. Showcasing work as a teacher, developer, and student with smooth animations and responsive design.

## Features

- **Bilingual Support**: English and Czech translations
- **Responsive Design**: Optimized for desktop and mobile devices
- **Smooth Navigation**: Hash-based single-page navigation with scroll animations
- **Animated Typography**: Dynamic text effects using react-typed
- **Modern Stack**: React 18 with hooks-based architecture

## Quick Start

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Available Scripts

- **`npm start`** — Dev server (localhost:3000)
- **`npm run build`** — Production-ready build
- **`npm test`** — Run tests with Jest

## Project Structure

```
src/
├── pages/           # Home, About, Projects, Contact sections
├── components/      # TopBar, Navigation, LanguageSelector
├── styles/          # CSS for pages and global styles
├── i18n/            # Translation files (EN/CS)
└── App.js           # Root component
```

## Sections

- **Home**: Introduction with animated text
- **About**: Profile information and skills
- **Projects**: Showcase of work and projects
- **Contact**: Get in touch section
